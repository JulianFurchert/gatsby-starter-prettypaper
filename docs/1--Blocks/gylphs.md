---
title: "Glyphs"
description: "The glyphs block is useful to show the characters of a custom font."
---

##  Example

example with all props
````
```glyphs
family: Open Sans
weight: 400
style: italic
characters: O,P,E,N
```
````

```glyphs
family: Open Sans
weight: 400
style: italic
characters: O,P,E,N
```

example with minium props
````
```glyphs
family: Open Sans
weight: 400
```
````

```glyphs
family: Open Sans

```
